# Avatar Images

Place your HR interviewer avatar image here as `hr-professional.jpg`

## Requirements
- Format: JPG or PNG
- Size: 512x512 or larger
- Content: Professional headshot, front-facing
- Quality: Clear, well-lit

## Example
Copy any professional photo here and rename it to:
- `hr-professional.jpg`

If no image is provided, the app will use a fallback cartoon avatar from DiceBear.
